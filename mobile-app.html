Aplicativo mobile - Controle de estoque

Do que se trata?

Esse projeto foi desenvolvido para uma empresa canadense de móveis com mais de 10 anos de experiência na indústria.

Uma das principais funcionalidades do aplicativo é a impressão de etiquetas diretamente do sistema, proporcionando uma maneira eficiente e precisa de identificar os produtos armazenados. Além disso, foi implementada uma tela de login personalizada, que permite identificar o usuário responsável por cada operação, fornecendo informações detalhadas sobre quem realizou a contagem de estoque.

No desenvolvimento do front-end, foram utilizados Svelte e o consumo de APIs REST para conectar o aplicativo ao banco de dados e garantir uma comunicação ágil e segura. Além disso, o aplicativo conta com algumas funcionalidades que possuem leitura de QR code, tornando o processo de leitura e rastreamento das tags ainda mais rápido e intuitivo.

Com esse aplicativo, a empresa agora tem total controle sobre o estoque, podendo identificar de forma precisa os produtos, as operações realizadas e até mesmo os operadores envolvidos. Essa solução moderna e eficiente otimizou o fluxo de trabalho, reduziu erros e trouxe maior transparência para todo o processo de controle de estoque.
